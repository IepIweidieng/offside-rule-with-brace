# Offside Rule with Brace

There are 5 different indentation tokens:
- `FRONT_INDENTED`
    - Increases indentation level
- `NO_INDENTED`
    - Do not affect indentation level
- `BACK_INDENTED`
    - Decreases indentation level
- `SEMI_NO_INDENTED`
    - Do not affect indentation level
- `SEMI_BACK_INDENTED`
    - Decreases indentation **width**
